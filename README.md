## Viewing the Presentation

This presentation is built for **[Deckset](https://www.deckset.com/)** — a Mac application that renders Markdown files as beautifully formatted slide decks.

### Option 1: Deckset (Recommended)

1. Download [Deckset](https://www.deckset.com/) (free trial available)
2. Open `presentation.md` in Deckset
3. The slides will render automatically with the **Ostrich** theme
4. Use **Presenter Mode** (⌘P) to see speaker notes during the presentation

### Option 2: Read as Markdown

The `presentation.md` file is fully readable as plain Markdown in GitHub or any text editor. Slide breaks are marked with `---` and speaker notes begin with `^`.

### Option 3: PDF Export

If you have Deckset installed, you can export to PDF via **File → Export → PDF**.
