# How to Push This Repo to GitHub

## Step 1: Create the repo on GitHub

Go to https://github.com/organizations/Plerk-code/repositories/new and create a new repo called `bitventure-case-study` (public or private).

**Do NOT** initialise it with a README — it already has one.

## Step 2: Push from your terminal

Open Terminal, navigate to this folder, and run:

```bash
cd ~/path/to/Bit\ Venture\ Interview/bitventure-case-study-repo
git remote add origin https://github.com/Plerk-code/bitventure-case-study.git
git push -u origin main
```

## Step 3: Share with BitVenture

Send this link to the panel:

```
https://github.com/Plerk-code/bitventure-case-study
```

The README contains full instructions for viewing the presentation in Deckset or as Markdown.
